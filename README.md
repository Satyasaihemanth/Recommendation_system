# 🎧 Music Recommendation System

## 📌 Overview
This project recommends songs based on user preferences using similarity-based recommendation techniques.

## 🚀 Features
- Personalized music recommendations
- Fast similarity search
- Simple UI

## 🛠 Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit / FastAPI

## 📂 Project Structure
- app.py → Main application
- dataset.csv → Music dataset

## ⚙️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run app:
streamlit run app.py

## 🎯 Output
Recommended songs based on user input 🎶
